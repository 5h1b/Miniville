- KadoKado Games (2004-2010)
- Dinoparc (2005)
- Socratomancie (2005)
- Dinocard (2006)
- Hammerfest (2006)
- Pioupiouz (2006)
- Miniville (2007)
- Alphabounce (2007)
- CafeJeux (2007)
- DinoRPG (2007)
- Hordes (2008)
- SkyWar (2009)
- Muxxu Intrusion (2009)
- Muxxu Kingdom (2009)
- Muxxu Fever (2010)
- Muxxu Kube (2010)
- Muxxu La Brute (2010)
- Muxxu Snake (2010)
- Naturalchimie 2 (2010)
- Muxxu CroqueMotel (2011)
- Muxxu Odyssey (2011)
- La Bourinette (2011)
- Mush (2012)
- Galaxy55 (alpha, 2012)
- Teacher Story (2013)
- Arkadeo Games (2013-2014)
- Uppercup Football (2014)
- Rockfaller Journey (2015)
- Monster Hotel (2015)
